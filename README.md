# my-portfolio# 
🛡️ Cybersecurity Portfolio & Security Dashboard

A terminal-inspired, high-performance portfolio built with **React** and **Vite**. This project showcases my technical journey in vulnerability assessment and network security, featuring interactive simulations and tool documentation.

## 🚀 Key Features
- **Integrated CLI:** A functional terminal component to explore project files and credentials.
- **WebHound Showcase:** Documentation for my automated web crawler and SQLi/XSS hunting tool.
- **Sentinel OSINT:** Insights into network reconnaissance and security tracking.
- **Cyber-Themed UI:** A sleek, dark-mode interface optimized for a developer aesthetic.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, CSS-in-JS
- **Typography:** JetBrains Mono & Syne
- **Design:** Modern, high-contrast security-focused UI
